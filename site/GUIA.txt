Para testar o funcionamento do site fazer o seguinte passo a passo: 

-> Instalar o Node.js na sua máquina
-> Abrir a pasta "/site..." no seu terminal
-> Escrever no terminal: 
    -> npm install -g nodemon
    -> npm start
    -> npm run dev
-> Abra a pasta index.html no VSCODE com a extensão Live Server
OU
-> Abra o navegador e acesse 'http://localhost:3000'.
-> Se tudo correr bem, o site deverá estar funcional :)

LEMBRETES AO USUÁRIO: 
->Se estiver rodando os arquivos via LiveServer, toda vez que for finalizar um registro de gastos do mês, reinicie o Live
  Server para o redirecionamento correto no 3º mês. Dessa forma será simulado o ato de fechar e abrir a aba numa ocasião
  convencional.
